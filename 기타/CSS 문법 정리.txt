링크방식: html에 링크를 삽입하여 외부 css 문서를 불러옴, 주로 사용(추천) / 병렬 호출

@기본선택자
1. 전체선택자(Universal Selector, *): 모든 태그에 적용
*{
   color: red;
} 
2. 태그 선택자(Type Selector, E): 태그 이름으로 요소 선택
li {
   color: red;
}
3. 클래스 선택자(Class Selector): HTML class 속성의 값으로 요소 선택, 주로 사용됨
.orange {
   color: red;
}
<li class="orange">오렌지</li>
4. 아이디 선택자(ID Selector)
#orange {
   color: red;
}
<li id="orange" class="orange">오렌지</li>

*클래스는 동일한 이름으로 여러 요소에 사용 가능하나, 아이디는 HTML 내 동일한 이름 중복 사용 불가

복합선택자
1. 일치 선택자(Basic Combinator): 두 개 이상의 기본 선택자의 조건을 모두 만족하는 요소 선택
span.orange {
}
<span class="orange">오렌지</span>
2. 자식 선택자(Child Combinator): 특정 선택자의 자식 요소를 선택
ul > .orange {
}
<ul>
  <li class="orange">오렌지</li>
3. 후손(하위) 선택자(Descendant Combinator): 특정 선택자의 후손, 하위 요소 선택
* 띄어쓰기로 구분 가능
div .orange{  // div 태그 내 클래스가 orange인 요소 선택 
}
<div>
 <ul>
  <li class="orange">오렌지</li>
 </ul>
</div>
div 태그 입장에서 ul 태그는 자식이며 li는 후손이다 

4. 인접 형제 선택자: 특정 태그의 형제 요소 중 다음에 인접해 있는 요소 하나를 선택
.orange + li {
}
<ul>
<li>수박
<li class="orange"
<li>망고  <-- 선택됨

5. 일반 형제 선택자: 특정 태그의 다음의 형제 요소 모두 선택
.orange ~ li {}
<ul>
<li>수박
<li class="orange"
<li>바나나 <-- 선택됨
<li>망고  <-- 선택됨


@가상 클래스 선택자(Pseudo-Classes Selectors)
일반적으로 : 표시가 있으면 가상 클래스 선택자

1. HOVER: 특정 요소에 마우스 포인터가 올라가 있는 동안에만 선택 
<a href="~~~">
a:hover {}

2. ACITVE: 특정 요소에 마우스로 클릭하고 있는 동안 선택
<div class=>
.div:active

3. FOCUS: 포커스 된 동안에만 선택, 대화형 콘텐츠에서만 사용(input 태그 대표)
E:focus

4. First Child: 특정요소가 형제 요소 중 첫번째 요소라면 선택
E:first-child

.fruits li:first-child{}
-> class가 fruits인 태그의 후손 요소 중 li 태그 요소의 첫번째 태그를 선택

. last child, nth-child(n)
.fruits li:nth-child(2){}
*** 해석은 오른쪽에서 왼쪽으로=>특정 태그의 2번째 자식 요소 선택하는데, 선택되는 태그는 li태그이며 동시에 id가 fruits인 태그의 하위임

5. nth-of-type(n): 특정 태그와 동일한 타입인 형제 요소 중 특정 태그가 n번째 요소라면 선택

6. 부정선택자: 특정 태그를 제외한 나머지 선택
.fruits li:not(.strawberry){}
=> 아이디가 strawberry인 선택자를 제외한 나머지 li 형제 태그들을 선택하는데, 그 형제태그들은 아이디가 fruits인 태그의 후손들임


@ 가상 요소 선택자(Pseudo-Elements Selectors): css 안에서 html 가상요소 생성
가상클래스는 : 한개 붙고, 가상요소는 :: 두 개 붙는다
1. Before: 특정 태그 안쪽 내용의 왼쪽에 특정 요소 추가 삽입
ul li::before { 
  content: "숫자";
}
=>li 태그 내 내용의 왼쪽에 "숫자"삽입하는데 그 태그들은 ul 태그의 하위임
*** 가상 요소 선택자 내 content를 작성하지 않으면 다른 특징들이 적용되지 않음.
특정 내용을 content에 넣지 않더라도 "" 처리해야 다른 특징이 적용됨
